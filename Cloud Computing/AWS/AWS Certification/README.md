# AWS certifications

AWS currently offers 12 certifications: a foundational certification, three associate-level certifications, two professional-level certifications, and six specialty certifications. You can find a full overview of all Learning Paths to prepare for the certifications on the Cloud Academy’s AWS Certifications page.

1. AWS Certified Cloud Practitioner (CLF-C01)
1. AWS Certified Developer – Associate (DVA-C02)
1. AWS Certified SysOps Administrator – Associate (SOA-C02)
1. AWS Certified Solutions Architect – Associate (SAA-C03)
1. AWS Certified DevOps Engineer – Professional (DOP-C02)
1. AWS Certified Solutions Architect – Professional (SAP-C02)
1. AWS Certified Advanced Networking – Specialty (ANS-C01)
1. AWS Certified Security – Specialty (SCS-C01)
1. AWS Certified Machine Learning – Specialty (MLS-C01)
1. AWS Certified Database – Specialty (DBS-C01)
1. AWS Certified Data Analytics – Specialty (DAS-C01)
1. AWS Certified SAP on AWS – Specialty (PAS-C01)
